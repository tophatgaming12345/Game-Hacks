# Game Hacks


## Farming Sim 19
Tophat’s FS19 Hacks is a lightweight save editor for Farming Simulator 19 that automatically locates your save folder and lets you modify your selected save slot in seconds. It allows you to set a custom money amount, remove all farm debt, and claim ownership of every field by editing the game’s XML save files safely (with automatic backups). The tool features a simple interface, works with both standard and OneDrive Documents locations, and requires the game to be closed before applying changes.
